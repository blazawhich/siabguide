# Install Eraser

Installing **Eraser** is an easy and quick process. To begin installing **Eraser**, perform the following steps:

As described in the How-to Booklet chapter [**6. Destroying Sensitive Information**](https://securityinabox.org/chapter-6)[Sensitive Information], **Eraser** wipes data from your hard disk by overwriting it with random information. The more times you overwrite the data, the less likely that it will be recovered.

## Step 1

**Double click** ![](https://securityinabox.org/sbox/screen/eraser-en/01.png)[Eraser img 01]; the *Open File - Security Warning* dialog box may appear. If it does, **click** ![Run](https://securityinabox.org/sbox/screen/eraser-en/02.png)[Eraser img 02] to activate the *InstallAware Wizard*; after some moments, the *Welcome to the InstallAware Wizard for Eraser* screen will appear.

## Step 2

**Click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] to activate the *License Agreement* screen, and then **click** checkbox to enable the *I accept the terms of the license agreement* option, and then **click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] again to activate the *Important Information* window.

## Step 3

**Click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] after reading the contents displayed in the scrolling window to activate the *Destination Folder* window and then **click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] again.

## Step 4

**Click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] to activate the following screen:

![Select Program Folder](https://securityinabox.org/sbox/screen/eraser-en/04.png)[Eraser img 04]

*Figure 1: The Select Program Folder window*

## Step 5

**Check** the *Only for me (current user)* option to ensure that only you are permitted to use **Eraser**, and then **click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] to activate the *Completing the InstallAware Wizard for Eraser* window.

## Step 6

**Click** ![Next](https://securityinabox.org/sbox/screen/eraser-en/03.png)[Eraser img 03] and then **click** ![Finish](https://securityinabox.org/sbox/screen/eraser-en/05.png)[Eraser img 05] to complete the installation process, and to run **Eraser** as follows:

![Eraser main user interface](https://securityinabox.org/sbox/screen/eraser-en/06.png)[Eraser img 06]

*Figure 2: The Eraser main user interface*
