# Install Pidgin and OTR

Both **Pidgin** and its associated **Off-the-Record** (**OTR**) automated encryption and authentication plugin must be installed properly for either program to work. Fortunately, the installation process for both the programs is easy and quick.

## Install Pidgin

### Step 1

**Double click** ![installer](https://securityinabox.org/sbox/screen/pidgin-en-1/01.png)[Pidgin img 01]; the *Open File - Security Warning* dialog box may appear. If it does, **click** ![Run](https://securityinabox.org/sbox/screen/pidgin-en-1/02.png)[Pidgin img 02] to display the following screen:

![Install Language confirmation box](https://securityinabox.org/sbox/screen/pidgin-en-1/03.png)[Pidgin img 03]

*Figure 1: The Install Language confirmation box*

### Step 2

**Click** ![OK](https://securityinabox.org/sbox/screen/pidgin-en-1/04.png)[Pidgin img 04] to display the *Welcome to the Pidgin 2.10.9 Setup Wizard* screen.  

### Step 3

**Click** ![Next](https://securityinabox.org/sbox/screen/pidgin-en-1/05.png)[Pidgin img 05] to display the *License Agreement* screen; after you have read the *License Agreement*, **click** ![Next](https://securityinabox.org/sbox/screen/pidgin-en-1/05.png)[Pidgin img 05] to display the *Pidgin 2.10.9 Setup - Choose Components* window.

### Step 4

**Click** ![Next](https://securityinabox.org/sbox/screen/pidgin-en-1/05.png)[Pidgin img 05] to display the *Pidgin 2.10.9 Setup - Choose Install Location* window. 

### Step 5

**Click** ![Install](https://securityinabox.org/sbox/screen/pidgin-en-1/06.png)[Pidgin img 06] to accept the default installation path, and display the *Pidgin 2.10.9 Setup - Installing* window to begin installing the **Pidgin** software.

*A number of folders and files will begin installing themselves in rapid succession; after the installation process has been completed, the Pidgin 2.10.9 Setup - Installation Complete window will appear.*

### Step 6

**Click** ![Next](https://securityinabox.org/sbox/screen/pidgin-en-1/05.png)[Pidgin img 05] to display the *Completing the Pidgin 2.10.9 Setup Wizard*.

**Note**: During ### Step 3** of the installation process, **Pidgin** was configured to be included in the **Start > All Programs** list, and can be launched from there in the future. 

### Step 7

**Click** ![Finish](https://securityinabox.org/sbox/screen/pidgin-en-1/08.png)[Pidgin img 07] to complete installing **Pidgin**.

##Â Install the Off-The-Record (OTR) Plugin

### Step 1

**Double click** ![OTR installer](https://securityinabox.org/sbox/screen/pidgin-en-1/09.png)[Pidgin img 08]; the *Open File - Security Warning* dialog box may appear. If it does, **click** ![Run](https://securityinabox.org/sbox/screen/pidgin-en-1/02.png)[Pidgin img 02] to display the following screen:

![Welcome to the pidgin-otr Setup Wizard](https://securityinabox.org/sbox/screen/pidgin-en-1/10.png)[Pidgin img 09]

*Figure 2: The Welcome to the pidgin-otr 4.0.0-1 Setup Wizard* 

### Step 3

**Click** ![Next](https://securityinabox.org/sbox/screen/pidgin-en-1/05.png)[Pidgin img 05] to display the *License Agreement* screen; after you have completed reading the *License Agreement*, **click** ![I Agree](https://securityinabox.org/sbox/screen/pidgin-en-1/11.png)[Pidgin img 10] to display the *pidgin-otr 4.0.0-1 Setup - Choose Install Location* screen.  

### Step 4

**Click** ![Install](https://securityinabox.org/sbox/screen/pidgin-en-1/06.png)[Pidgin img 06] to begin the installation process.

### Step 5

**Click** ![Finish](https://securityinabox.org/sbox/screen/pidgin-en-1/08.png)[Pidgin img 07] to complete installing the **Pidgin-OTR** messaging software plugin.

After you have completed installing both **Pidgin** and **OTR**. The following icon will appear in the Windows task bar when Pidgin is started from the **Start > All Programs** list:

![Pidgin-OTR icon outlined in black](https://securityinabox.org/sbox/screen/pidgin-en-1/12.png)[Pidgin img 11]

*Figure 3: The Pidgin-OTR icon outlined in black*

Congratulations! You have successfully completed installing both the **Pidgin** and **OTR** programs!
